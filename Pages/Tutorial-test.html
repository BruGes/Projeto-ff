Visual Studio Code - Tutorial 1

Vscode é um editor de código desenvolvido ela Microsoft que contém diversas funcionalidades e suportes como depuração, controle de versionamento Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código.

Ele é um dos editores mais populares, por seu amplo suporte e suas extensões que podem ser baixadas por ele mesmo em uma aba especifíca de extensões, sendo também quase que completamente, senão completamente customizado.

Existem diversas atalhos internos do Vscode que ajudam a digitar de forma mais eficaz e agíl, como também de testar seu código. Ele também mostrara as possíveis escolhas para o código que você está escrevendo e caso você concorde ou não lembre o código completo, apertando nele ele autocompletara seu código.

E como eu baixo ele?

O Vscode tem um site próprio onde consegue-se baixar todas as versões dele, nós recomendamos baixar aquelas onde se tem escrito Stable Build que significa que é uma versão estável e está melhor otimizado para o Windows ou o sistema operacional que se está utilizando
